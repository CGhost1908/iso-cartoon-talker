ISO Cartoon Talker Kurulumu

ISO Cartoon Talker'ı bilgisayarınıza kurmak için lütfen aşağıdaki adımları izleyin:

  1. 1- Kurulum dosyaları

    C:\Program Files

    dizinine kaydedilecektir.
  2. 2- Kurulum için

    en az 50GB

    boş yer olmalıdır.
  3. 3- Yaklaşık 25GB veri indirilecektir. Kablosuz ağa bağlı olduğunuzdan emin olun.
  4. 4-

    Python 3.10 (3.10.9 sürümünde test edildi) sürümü yüklü ve Path'de etkin olmalıdır.

  5. 5- Uygun Python sürümü yüklü olmadığı takdirde hata alacaksınızdır.

  6. 5- Git yüklü ve Path'e ekli olmalıdır.

  7. 6- Setup dosyasi

    yönetici olarak

    çalıştırılmalıdır.
  8. 7- "Install" butonuna tıklayın.
  9. 8- Kurulum tamamlandığında, masaüstünüzdeki kısayolu kullanarak programı başlatabilirsiniz.
